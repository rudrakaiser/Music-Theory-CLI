# Music Theory CLI

A simple **C++ Command-Line Interface (CLI) tool** to help musicians, students, and music enthusiasts explore **basic music theory** concepts like scales, triads, and chord families.

---

## Features

- Display **Major & Minor Scales** with intervals (Whole/Half steps)
- Display **Major & Minor Triads**
- Display **Major & Minor Chord Families**
- Interactive and **user-friendly CLI**
- Modular, easy-to-read C++ code

---

## Installation

1. Clone this repository:

    ```bash
    git clone https://github.com/rudrakaiser/Music-Theory-CLI.git
    ```
2. Navigate to the folder:
    ```bash
    cd Music-Theory-CLI/src
    ```
3. Compile the code (example using g++):
    ```bash
    g++ main.cpp -o main
    ```
4. Run the program:
    ```bash
    .\main
    ```

## Usage

The program provides a menu for - Major/Minor **Scales**, **Triads** and **Chord Families**.

> Simply enter the number of your choice and follow the prompts.

```
------------| MUSIC THEORY CLI |------------

1. MAJOR SCALE
2. MINOR SCALE
3. MAJOR TRIADS
4. MINOR TRIADS
5. MAJOR CHORD FAMILY
6. MINOR CHORD FAMILY
7. ABOUT
0. EXIT

Enter Choice >>> _
```
